# GPT.md

## 프로젝트

- 이름: 에이든 전투 시뮬레이터
- 유형: 데스크톱 전용 단일 화면 정적 웹앱
- 목적: 이터널 리턴 실험체 에이든 조작·스킬 콤보 연습
- 기술: HTML, CSS, Vanilla JavaScript
- 프레임워크·번들러·외부 의존성·테스트·린트·CI: 없음
- 언어: 한국어 UI

## 문서 기준

1. 실행 동작: `app.js`
2. DOM 계약: `index.html`
3. 시각 표현: `styles.css`
4. 조작감 설정: `control.md` JSON 블록
5. 구조 설명: `CLAUDE.md`
6. 변경 이력: `1.2_patch.md`

문서와 코드 불일치 시 실행 코드 우선. 변경 후 관련 문서 동기화.

## 핵심 파일

- `index.html`: 필드, 월드, 사이드 패널, 스킬 독, 사거리 인디케이터, 시전 바
- `app.js`: 상태, 입력, 게임 루프, 카메라, 이동, 판정, 스킬, 이펙트, 음성, 렌더링
- `styles.css`: 레이아웃, 원근 필드, HUD, 스킬 슬롯, 도형 이펙트, 애니메이션
- `control.md`: 런타임 조작감 설정 JSON
- `src/`: 스킬 아이콘, 캐릭터 스프라이트, 콘셉트 아트, 음성
- `skill effect sample/`: GIF 이펙트 참고 자료, 런타임 미사용

## 실행

```powershell
python -m http.server 8000
```

- URL: `http://localhost:8000`
- `file://` 비권장: `control.md` fetch, 오디오 재생 제한
- 최소 화면 너비: 980px

## 런타임 구조

- 단일 전역 상태: `state`
- 루프: `requestAnimationFrame(tick)`
- 처리 순서: 캐스트 → 타이머 → 이동 → 더미 → 카메라 → 렌더링
- 렌더링: `renderUnits`, `renderRanges`, `renderPanel`, `renderCast`, `renderSkills`
- 더미 DOM: 매 프레임 `innerHTML` 재생성
- 스킬 효과: 임시 DOM 생성, CSS `@keyframes`, 타이머 제거
- 캐스트 큐: `state.casts`
- 입력 버퍼: `state.buffer`, 최근 입력 1개
- 재시전 타이머: `state.recast.W/E/R`

## 좌표·카메라

- 게임 좌표: 필드 기준 0~100 퍼센트
- 거리·사거리·충돌·이펙트: 픽셀
- 변환: `toPx()`, `fromPx()`
- 월드 배율: `WORLD_SCALE = 10`
- CSS `.ground` 크기와 `WORLD_SCALE` 일치 필수
- 거리 기준: `M = RANGE.E1 / 6.5`
- 카메라: `perspective + rotateX + scale + translate`
- 카메라 상수: `CAMERA.fovDeg`, `depressionDeg`, `zoom`
- 입력 좌표 역투영: `pointFromEvent()`
- 카메라 잠금: 캐릭터 추적
- 잠금 해제: 화면 가장자리 패닝
- Space: 일시 캐릭터 고정

## 조작

- 우클릭 빈 필드: 이동
- 우클릭 더미: 사거리 내 평타, 사거리 밖 추격 평타
- Shift+클릭: 강제 평타, 이동 없음
- Q/W/E/R/D/F: 스킬
- S: 이동·추격·버퍼 이동 정지, 스킬 돌진 유지
- C: 쿨타임 초기화 토글
- F11: 전체화면 전환, Esc: 전체화면 종료; 두 액션 모두 조작키 변경 지원
- A: 평타 사거리 유지 표시
- A 사거리 표시 활성 + 원 내부 좌클릭: 이동 없이 사거리 내 최근접 적 평타
- A 사거리 원 외부 좌클릭: 공격 없음
- Esc: 인디케이터 조준 취소
- 키 바인딩: `state.keybinds`, 사이드 패널 `조작키 변경` 버튼, `#keybindModal` 재지정 모달; `FULLSCREEN`·`EXIT_FULLSCREEN` 포함
- 쿨타임 초기화 액션: `COOLDOWN_RESET`, 기본 `KeyC`
- 스킬 후 자동 공격: 스킬·돌진·W 충전 종료 후 사거리 내 최근접 더미 연속 평타, 이동·정지·수동 공격 명령·사거리 이탈 시 종료
- 사거리 표시 OFF: 즉시 시전
- 사거리 표시 ON: 키 누름 조준, 키 해제 또는 우클릭 시전
- W: 조준 홀드 예외, 즉시 충전 시작
- 스킬 시전 방향·R1 위치: 시전 시작 시 고정
- 몸 기준 효과 원점: 시전 완료 시점의 에이든 위치

## 에이든 상태기계

- 일반 상태: 근접 평타, 뇌격 Q
- 근접 평타: 히트스캔, 투사체 없음, 공격 프레임 즉시 적중
- 과전하 원거리 평타: 투사체 유지
- 전하: 최대 5, 획득 후 5초 유지, 추가 획득 시 갱신
- 전하 만충: E 초기화, 주변 적 조건 또는 E 백스텝으로 과전하 진입
- 과전하: 7초, 탄환 5개, 원거리 평타, 전자포 Q
- 과전하 종료: 시간 만료 또는 탄환 소진
- 종료 보상: 2초간 이동 속도 +13%
- 원거리 평타: 탄환 1개 소모
- 원거리 Q: 탄환 1개 소모
- 일반 Q·원거리 Q: 독립 쿨다운 `Q_NORMAL`, `Q_OVER`
- 평타 적중: 현재 폼 Q 쿨다운 1.5초 감소

- 평타 속도 제한: `state.nextBasicAttackAt` 독립 잠금, 간격 `1 / attackSpeed()`, 쿨타임 초기화 모드 영향 없음
- 전하 충전 버튼: 일반 상태 `addCharge()`로 `chargeDecay` 동시 갱신, 과전하 상태 탄환·지속시간 재충전
- 적중 사운드: W=`W_타격.mp3`, E=`E_타격.mov`, R=`R_타격.mov`, 원거리 평타=`평타_원거리.mp3`, 원거리 Q=`Q_원거리_타격.mov`, 근거리 Q=`Q_타격.mov`, 근거리 평타·D=`타격.mov`

- `control.md`의 첫 `json` 코드블록 로드
- 파싱 실패: `CONTROL_DEFAULTS` 사용
- 주요 설정: 이동 속도, 도착 반경, 카메라 추적, 줌, 입력 버퍼, 타겟 클릭 반경, 쿨다운, 디버그
- 스킬 고유 사거리·시전시간·재시전 구조: `app.js` 우선
- 제네릭 `castDelay/activeTime/recovery`: 미사용
